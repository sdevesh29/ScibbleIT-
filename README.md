# ScibbleIT-
# Installing Files
Download the files.  
Put files in C:/Users/your_username  
install node.js on your system.  

# After Installing
run npm  
on npm use commands:  
  cd ScribbleIT  
  npm install  //this is to install node modules  
  npm start  
  
Now the server will be listening to a port. (8080)  
Run localhost:8080 on your default browser.
